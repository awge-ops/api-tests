Название проверки,Ожидаемый результат
GET /users?gender=male возвращает список пользователей,Ответ содержит массив ID пользователей
GET /users?gender=female возвращает список пользователей,Ответ содержит массив ID пользователей
GET /users?gender с некорректным значением,Ошибка (400 или аналогичная)
GET /user/{id} с существующим id,Ответ содержит данные пользователя
GET /user/{id} с несуществующим id,Ошибка (404 или аналогичная)
GET /user/{id} с некорректным id (строка, спецсимволы),Ошибка (400 или аналогичная)
Поля пользователя: id, name, gender, age, city, registrationDate,Все обязательные поля присутствуют и имеют корректные типы
